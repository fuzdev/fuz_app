<T>(c: Context<any, any, {}>, value: string, options: SessionOptions<T>): void import {set_session_cookie} from '@fuzdev/fuz_app/auth/session_middleware.js';

Set the session cookie on a response.

options.max_age is the single source of truth for cookie lifetime: it drives both the embedded expires_at (via create_session_cookie_value) and the cookie's HTTP Max-Age attribute set here. Falls back to session_cookie_options.maxAge (= SESSION_AGE_MAX) when unset. options.cookie_options cannot carry maxAge (omitted in the type) so the two values can't drift.
